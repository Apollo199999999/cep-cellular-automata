About

This is a modified version of Conway's Game of Life, with the following rules:

  • Birth: A dead cell with exactly three live neighbours becomes 100% alive (reproduction).
  • Survival: A live cell with exactly three live neighbours stays 100% alive (stasis).
  • Overpopulation: If a live cell has 4 neighbours, it has an 80% chance of dying, and will be 20% alive if it lives. If a live cell has 5 or more neighbours, it will die.
  • Loneliness: If the cell has 2 neighbours, it has a 20% chance of dying, and will be 80% alive if it lives. If the cell has 1 neighbour, it has an 80% chance of dying, and will be 20% alive if it lives. If the cell has 0 neighbours, it will die.
  • Appearance: The cell will be coloured based on how alive it is. A 100% alive cell will be completely black, while a 0% alive cell will be completely white.

This game also generates music using the dead/alive cells. For every column in the 2D grid, each row will be mapped to a note in the D major pentatonic scale, and for each row, if the row contains an alive cell, that row's corresponding note is played, which will cause each column that contains alive cells to produce a chord. To play the sound, click the corresponding buttons under the "Controls" section. Do note that this will pause the game at the current generation.

How to use:

  • Use the "Controls" section to control the game.
  • To make a cell alive, pause the game and click on a dead cell to make it become alive.
  • You can generate music using the dead/alive cells by clicking the corresponding buttons under the "Controls" section.
